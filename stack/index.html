Operating Notes

How I choose.

Start boring

Static HTML, plain CSS, and vanilla JavaScript are still excellent when the shape of the problem is simple.

Make loops visible

Games and tools both benefit from obvious state, deterministic update loops, and small modules with honest names.

Ship the path

Deployment and hosting count as product work. A feature is not done until someone can reach it without a local setup ritual.
